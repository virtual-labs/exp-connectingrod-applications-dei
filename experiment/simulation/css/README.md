# Interactive Connecting Rod Simulation

This directory contains the interactive simulation page for the Connecting Rod experiment.

## Features
- **Interactive Navigation**: Users can select one of four specific simulations using the play buttons at the top.
- **Dynamic Content**: Only the active simulation and its description are visible at any given time to prevent page clutter.
- **Auto-Pause**: Switching between simulations automatically pauses the hidden video to conserve resources.

## Files
- `index.html`: The main interface with the navigation logic.
- `main.css`: Styles for the custom buttons and responsive layout.
